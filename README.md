# CL-Class ohjekirja

Mercedes-Benz CL-Class (2008) käyttöohje suomeksi (769 sivua).

**https://thebig8.github.io/cl-class-ohjekirja/**

## Käyttö

- Haku
- Suomi | Alkuperäinen (sama sivunumero)
- Sivuvalitsin 1–769

## Paikallinen kehitys

```bash
python3 -m http.server 8080
```

Avaa http://127.0.0.1:8080 (älä avaa `index.html`-tiedostoa suoraan).
